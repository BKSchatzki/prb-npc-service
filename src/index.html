PartyRoomBloom NPC Service API

  • /api/v1/names

    Displays a complete list of all names, sorted by species, given name (by gender), and surname.

  • /api/v1/traits

    Displays a complete list of all traits:

    gender, age, anatomy (build, height), skin (color, texture), hair (color, style), eyes (color, shape), voice (pitch, quality, speed, volume), distinguishing features, mannerisms, core motivations, plot hooks.

  • /api/v1/npc

    Creates one totally randomized NPC with one of each trait.

  • /api/v1/npc/:species

    Creates one random NPC of a chosen species with one of each trait. Valid species are: aasimar, dragonborn, dwarf, elf, gnome, goliath, halfling, human, orc, tiefling.
